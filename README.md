# TODO CLI

### Description
A simple but functional CLI application to manage your TODO list.

- Features
  - Add TODO
  - Remove TODO
  - Mark TODO item as done
  - List all TODO items

# Compile
```bash
go build -o todo-cli cmd/main.go
```
